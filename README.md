# Rock Paper Scissors Game Using Python

## 📌 Description
This project is a Rock Paper Scissors game developed using Python. 
The user plays against the computer by choosing Rock, Paper, or Scissors, and the program randomly generates the computer's choice to decide the winner.

This project was created as part of my CodSoft Python Programming Internship tasks.

## 🚀 Features
- User vs Computer gameplay
- Random computer choice generation
- Displays game results (Win, Loss, or Draw)
- Tracks the number of wins, losses, and draws
- Simple command-line interface
- Interactive user input

## 🛠️ Technologies Used
- Python 3
- Random module
